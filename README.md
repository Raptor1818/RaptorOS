# RaptorOS

Made to emulate the feel of a desktop enviroment. 

## Features

### Window Functionalities
- Drag and resize
- Dynamic zIndex levels
- Close and minimize, with animations
### Taskbar
- Icon states change based on their windows' states
- Clock
### Much more to come!
